localsearch-inhibit(1)
======================

== NAME

localsearch-inhibit - Inhibit indexing temporarily

== SYNOPSIS

*localsearch inhibit* <__command__>
*localsearch inhibit* [--list | -l]

== DESCRIPTION

*localsearch inhibit* pauses the LocalSearch indexer while __command__ is
running.

If called with the *--list* option. The existing reasons to have the
indexer paused will be listed.

== SEE ALSO

*localsearch-status*(1), *localsearch-3*(1).
