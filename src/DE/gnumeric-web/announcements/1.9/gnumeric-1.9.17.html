Gnumeric 1.9.17 aka "TBD" is now available.

The Gnumeric Team is pleased to announce the availability of Gnumeric version 1.9.17. This release requires the concurrently released goffice 0.7.17.

This is a bug fix release in preparation for Gnumeric 1.10.

  • Andreas
    • Update EURO and EUROCONVERT with currencies added in 2007 to 2009.
    • Export hyperlinks to ODF. [Bug 603533].
    • Make EUROCONVERT Excel compatible [Bug 603755].
    • Update modification times. [Bug 603099].
    • Add RANK.AVG (rank function returning average rank for ties).
    • Added Sign Test to statistical analysis tools.
  • Gabriel Kerneis
    • Implement --one-file-per-sheet in ssconvert [Bug 604549].
  • Jean
    • Fixed problem with row headers. [Bug 603862].
    • Fixed pane origin when zoomed. [Bug 604373].
    • Make the cell comment red triangle visible in RTL mode. [Bug 604580].
  • Morten
    • Elminate use of "float".
    • Fix selection problem. [Bug 601107].
    • Automate solver testing.
    • Fix SUMIF problems. [Bug 603671] [Bug 604214].
    • Fix DATEDIF problem across bogus 1900-02-29. [Bug 599461].
    • Fix graph editor crash. [Bug 604223].
    • Fix goal seek problems.
    • Fix ODS import crash. [Bug 604615].
