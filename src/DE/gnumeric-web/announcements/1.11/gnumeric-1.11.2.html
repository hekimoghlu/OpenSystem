Gnumeric 1.11.2 aka "TBD" is now available.

The Gnumeric Team is pleased to announce the availability of Gnumeric version 1.11.2. This release fixes a number of mostly minor issues since the previous version.

  • Andreas
    • Colour the range expressions to match the range cursor. [Bug 632156].
    • Implement ctrl-click cell deselection. [Bug 610696].
    • Fix LaTeX export of cyrillic. [Bug 666340].
    • Fix function tool tip. [Bug 668685].
    • Delay function completion tooltip. [Bug 668683].
    • Import inline text from xlsx files. [Bug 668936].
    • Fix rich text import from xlsx. [Bug 669083].
    • Fix data shuffle. [Bug 669184].
    • Clarify the skew-t and skew-normal descriptions.
    • Fix EXACT. [Bug 670232].
    • Fix editing of decimals. [Bug 670558].
    • Fix edit line display. [Bug 555519].
    • Avoid recalculation on pure comment deletion. [Bug 670603].
  • Jean
    • Fix formula interpretation in plot series (with Morten's help). [Bug 662237].
    • Don't destroy a sheet object when inserting it. [Bug 669648].
  • Morten
    • Improve parsing on numbers with non-ascii digits.
    • Make solver check for exceeding the time limit.
    • Fix object life-cycle problems in solver.
    • Fix multihead problem with format dialog.
    • Fix auto-expression problem with sheet deletion.
    • Fix ellipsizing of auto-expression.
    • Fix auto-expression of single cell when moved.
    • Fix problems with underlines and rotated cells. [Bug 667152].
    • Fix multihead problem with auto-expressions.
    • Fix allocate-free mismatch in sheet management dialog.
    • Fix paste special undo problem. [Bug 667702].
    • Fix F4 (abs/rel-cycling) bug. [Bug 668625].
    • Fix minor function name completion bug. [Bug 668624].
    • Fix abs/rel stickiness in expression entry. [Bug 668691].
    • Minor quit-dialog improvements.
    • Put DataShuffle back on the menus.
    • Resurrect the splash screen from bit rot. [Bug 670449].
    • Make sheet removal respect manual recalculation. [Bug 671192].

goffice 0.9.2 aka "TBD" is now available.

The Gnumeric Team is pleased to announce the availability of goffice version 0.9.2.

  • Andreas
    • Improve sensitivity handling of SI related widgets in format selector.
    • Fix scientific format with SI prefix selection. [Bug 665301].
    • Fix recognition of formats with SI prefix by format selector. [Bug 665303].
    • Handle numeral shapes for number formats.
  • Jean
    • Fixed rounding errors in limits displayed in contour plot legend. [Bug 666292].
    • Support wrapped charts labels, partially fixes Bug 643873.
    • Fixed GdkPixbuf usage. [Bug 667005].
    • Don't truncate images when exporting a chart. [Bug 645938].
    • Add automatic Bezier spline interpolation. [663356].
    • Fixed graph type selector position. [Bug 667793].
    • Fix bounds for partial range logarithmic axes. [Bug 669257].
    • Fix Y-axis label position. [Bug 669325].
    • Add limits to the regression curve display range. [Bug 669252].
    • Don't move any other axis when changing an axis position. [Bug 669050].
  • Morten
    • Embed library ui files into library.
    • Embed svg-patterns.xml and error bar image files too.
    • Embed icons and ui files into plugins.
