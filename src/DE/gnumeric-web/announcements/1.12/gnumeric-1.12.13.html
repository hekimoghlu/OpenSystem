Gnumeric 1.12.13 aka "TBD" is now available.

The Gnumeric Team is pleased to announce the availability of Gnumeric version 1.12.13.

Noteworthy news in this release:

  • ods import/export export fixes.
  • xls import/export export fixes.

Details of changes by contributor:

  • Andreas
    • Fix export of comboboxes to ODF.
    • Fix export of help messages to ODF.
    • Import invisible alignment characters in number formats from ODF. [part of Bug 725852].
    • Fix fill-alignment export/import to and from ODF. [part of Bug 726035].
    • Fix distributed-alignment export to ODF. [part of Bug 726035].
    • Fix ODF roundtrip of "center accross selection". [Bug 726010].
    • Improve ODF roundtrip of conditional formats. [Bug 726201].
    • Fix import & export of cell indent from/to ODF.
    • Fix ODF roundtrip of underlines.
  • Jean
    • Fix tooltip position for scale or RTL sheets. [Bug 725941].
    • Really fix the hyperlink tips persistence. [see Bug 706659].
  • Morten
    • Support volatile functions. [Bug 305798].
    • Improve testing.
    • Import/Export super/subscript fonts to xlsx. [Part of Bug 726035].
    • Import xlsx auto-filters. [Bug 725460].
    • Update Gnumeric schema.
    • Fix rich text problem. [Bug 726086].
    • Improve xls import/export of conditional formats. [Bug 726202].
    • Fix case sensitivity for conditional formats. [Bug 726246].
    • Fix minor conditional format GUI issues.
    • Plug leak.
    • Fix font dialog problem with underline.
    • Fix crash reading strange xls file. [Redhat Bug 1076912].
    • Fix problem clearing existing values in some xlsx cells. [Bug 726297].
    • Fix criticals for dynamic deps in conditional formats. [Bug 726163].
    • Fix LOG2 problem. [Bug 726464].

goffice 0.10.13 aka "TBD" is now available.

The Gnumeric Team is pleased to announce the availability of goffice version 0.10.13.

Details of changes by contributor:

  • Andreas
    • Write invisible characters in ODF number style export. [part of Bug 725852].
    • Fix writing of scientific number format to ODF. [part of Bug 725852].
    • Fix strict number style export to ODF. [Bug 726237].
  • Jean
    • Ensure that the lasem plugin builds with lasem-0.4.1. [Bug 725693].
    • Support leave_notify events in the canvas. [Bug 668103].
  • Julian Sikorski
    • Remove extra DESTDIR in mathml to itex converter. [Bug 725684].
  • Morten
    • Fix problem saving print settings. [Bug 698630].
    • Plug leaks.
    • Fix FMW problem with GocPolyline/GocPolygon.
