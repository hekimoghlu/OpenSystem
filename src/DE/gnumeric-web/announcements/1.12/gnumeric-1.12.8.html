Gnumeric 1.12.8 aka "TBD" is now available.

The Gnumeric Team is pleased to announce the availability of Gnumeric version 1.12.8.

Noteworthy news in this release:

  • Win32 fixes
  • Andreas
    • Update/fix documentation [Bug 708679][Bug 707075][Bug 708556].
    • Fix paste into locked cells [Bug 702292].
  • Jean
    • Fix radio buttons behavior in covariance dialog. [Bug 707595].
    • Add a border around font selector in cell format dialog. [Bug 708567].
  • Morten
    • Fix fuzzed file crash. [Bug 707875].
    • Use lighter green for function markers. [Bug 708573].
    • Fix item cursor problem. [Bug 709320].
    • Fix win32 compilation.
