Gnumeric 1.12.15 aka "TBD" is now available.

The Gnumeric Team is pleased to announce the availability of Gnumeric version 1.12.15.

Noteworthy news in this release:

  • ods/xlsx graph import fixes
  • ods import/export fixes

Details of changes by contributor:

  • Andreas
    • Fix chart size in ODF import. [Bug 728278].
    • Improve chart roundtrip through ODF. [Bug 728197].
    • Export centre-across-selection in a way LO can understand. [Bug 726010].
    • Improve format roundtrip through ODF.
  • Morten
    • For xlsx, don't complain over "ext" elements.
    • xlsx graph import improvements.
    • Attempt a fix for tabs appearance with gtk+ 3.12. [Bug 728465].
    • Make it possible to have all callables as python functions. [Bug 728611].
    • Improve format roundtrip through ODF.
    • Plug leaks.
    • Make search dialog non-modal again. [Bug 728817].
