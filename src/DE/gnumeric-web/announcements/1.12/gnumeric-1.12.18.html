Gnumeric 1.12.18 aka "TBD" is now available.

The Gnumeric Team is pleased to announce the availability of Gnumeric version 1.12.18. This release requires the concurrently released goffice 0.10.18.

Noteworthy news in this release:

  • Workaround gtk+ api/abi breaks.

Details of changes by contributor:

  • Andreas
    • Improve documentation. [Bug 733466].
    • Add warning dialog. [Bug 733352].
    • Fix rich text roundtrip through ODF. [Bug 737039].
  • Jean
    • Fix dialogs behavior in full screen mode. [Bug 736395].
    • Save axis format to xls. [Bug 736656].
    • Fix recent files dialog crasher. [Bug 737261].
  • Morten
    • Fix xls export problem. [Bug 733771].
    • Update WEEKDAY function.
    • Fix xlsx export problem for strings. [Bug 736202].
    • Improve xlsx import of rich text.
    • Text suite improvements.
    • Fix minor ods rich text import problem.

goffice 0.10.18 aka "TBD" is now available.

The Gnumeric Team is pleased to announce the availability of goffice version 0.10.18.

Details of changes by contributor:

  • Jean
    • Fix grid line theme support. [Bug 733403].

    • Fix font color theme support. [Debian 757611].
    • Fix saving color maps when directory does not exists. [Bug 735008].
    • Fix color maps loading. [Bug 735007].
    • Don't pass NULL to g_strtod(). [Bug 735555].
  • Morten
    • Work around gtk+ ABI break for colour selector. [Bug 733350].
