Gnumeric 1.12.25 aka "TBD" is now available.

The Gnumeric Team is pleased to announce the availability of Gnumeric version 1.12.25. This is mainly a bugfix release.

Details of changes by contributor:

  • Jean
    • Add support in recent files dialog for multiple selesction and double click. [Bug 759160].
  • Morten
    • Fix TABLE problems. [Bug 752181] [Bug 752178] [Bug 752182].
    • Fuzzed file fixes. [Bug 752179] [Bug 752180].
    • Add R.DGUMBEL, R.PGUMBEL, R.QGUMBEL.
    • Fix conditional format problem. [Bug 750271].
    • Test also RANDWEIBULL and RANDLOGNORM.
    • Fix problem with R.QHYPER.
    • Add R.DRAYLEIGH, R.PRAYLEIGH, R.QRAYLEIGH.
    • Fix GUI translation problems.
    • Fix bugs with merged cells. [Bug 504004].
    • Make builds more reproducible.
    • Improve ssdiff for validation.
    • Fix problem with xls export of list validation. [Bug 557555].
