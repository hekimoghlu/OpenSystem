Gnumeric 1.12.16 aka "TBD" is now available.

The Gnumeric Team is pleased to announce the availability of Gnumeric version 1.12.16.

Noteworthy news in this release:

  • ODS import/export improvements

Details of changes by contributor:

  • Andreas
    • Fix database-range import from ODF. [Bug 730197].
    • Improve chart roundtrip through ODF. [Bug 728197].
    • Fix export of non-ODF interpolation types to ODF.
    • Fix export of titles and subtitles to ODF. [part of Bug 730397].
    • Fix axis sub-element order in ODF export. [part of Bug 730397].
    • Fix undo crash related to hyperlinks. [Bug 730466].
  • Morten
    • Start moving off GtkUIManager.
    • Clean out old #ifdef.
    • Fix ods series marker import/export.
    • Fix plan-perfect issue. [Debian Bug 748054].
    • Make installtion work without scrollkeeper.
    • Improve import/export testing. [Bug 730397].
    • Speed up certain large ranges of conditional formatting.
    • Work towards making ods export deterministic.

goffice 0.10.16 aka "TBD" is now available.

The Gnumeric Team is pleased to announce the availability of goffice version 0.10.16.

Details of changes by contributor:

  • Jean
    • Don't use GDK_THREADS_*. [728793].
    • Fixed Gog3DBox properties editor.
    • Add a view class to Gog3DBox to support data that are view dependent.
    • Restore dashes with dots in charts lines.
  • Morten
    • Clean out old #ifdef mess.
    • Improve axis value computation.
    • Improve xml import/export of GOStyle.
