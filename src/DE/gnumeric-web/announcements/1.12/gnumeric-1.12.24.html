Gnumeric 1.12.24 aka "TBD" is now available.

The Gnumeric Team is pleased to announce the availability of Gnumeric version 1.12.24.

This is a maintenance release. The only noteworthy fix is for printing of certain sheet objects.

Details of changes by contributor:

  • Jean
    • Fix printing of sheet objects with one cell or absolute anchors. [Bug 753961].
    • Fix snap to grid for one cell or absolute anchored objects. [Bug 754792].
    • On export, just change the extension, not more. [Ubuntu Bug 1497344].

goffice 0.10.24 aka "TBD" is now available.

The Gnumeric Team is pleased to announce the availability of goffice version 0.10.24.

This is a maintenance release.

Details of changes by contributor:

  • Jean
    • Fix bounds and distance for rectangle and paths in RTL mode. [Bug 753963].
    • Don't access NULL X or Y values in YXZ matrix plots. [Bug 754590].
