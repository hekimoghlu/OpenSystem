Gnumeric 1.12.28 aka "TBD" is now available.

The Gnumeric Team is pleased to announce the availability of Gnumeric version 1.12.28. This release requires the concurrently released goffice 0.10.28.

Noteworthy news in this release:

  • Solver sensitivity report
  • Complex number improvements

Details of changes by contributor:

  • Morten
    • Fuzzed file fixes. [Bug 761663] [Bug 761727] [Bug 762278].
    • Plug leaks.
    • Fix problems with ssconvert to lp/cplex formats.
    • Add sensitivity report to solver.
    • Fix bounds problem with cplex exporter.
    • Fix IMARCCOSH(1).
    • Add more tests for complex number evaluation.
    • Improve accuracy of IMLOG10.

goffice 0.10.28 aka "TBD" is now available.

The Gnumeric Team is pleased to announce the availability of goffice version 0.10.28.

Details of changes by contributor:

  • Jean
    • Fix histograms with missing values. [Bug 763446].
  • Morten
    • Configuration fixes.
    • Introspection improvements.
