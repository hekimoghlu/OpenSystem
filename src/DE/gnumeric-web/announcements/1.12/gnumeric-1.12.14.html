Gnumeric 1.12.14 aka "TBD" is now available.

The Gnumeric Team is pleased to announce the availability of Gnumeric version 1.12.14. This release requires the concurrently released Goffice 0.10.14. A recent Libgsf is recommended.

Noteworthy news in this release:

  • Many fixes related to conditional formats.
  • Try to disable Ubuntu's non-working scroll bars.

Details of changes by contributor:

  • Andreas
    • Improve format import from ODF files.
  • Morten
    • Plug leaks.
    • xlsx conditional format fixes. [Bug 726202].
    • Fix semantics of conditional format operators. [Bug 726806].
    • Fix xlsx quotes in formula strings. [Bug 726824].
    • Fix xls export of rich text.
    • Fix problems with large, circular farms of VLOOKUP etc. [Bug 723894].
    • Let goffice handle conditional number formats.
    • Attempt a workaround for Ubuntu scroll bars.
    • Fix odf parse problems.
    • Fix array editing.
    • Fix column/row header colouring problem. [Bug 728007].

goffice 0.10.14 aka "TBD" is now available.

The Gnumeric Team is pleased to announce the availability of goffice version 0.10.14.

Details of changes by contributor:

  • Andreas
    • Improve conditional format roundtrip through ODF. [part of Bug 725852].
  • Jean
    • Fix canvas drawing problem.
  • Morten
    • Work on conditional formats for ODF.
    • Implement numbered colours in number formats.
    • Fix plugin deactivation.
    • Plug leaks.
    • Fix rounding problem.
    • Use musl code for dtoa. Adapt to round-ties-away-from-0.
