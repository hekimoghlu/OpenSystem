# build.ps1 overrides the status of each test in this file when running the
# Swift Runtime tests for Android. These tests fail for Android if and only if
# built with the Windows toolchain and run through the Windows test driver.
# The set of failures is different on Linux. We cannot express this condition in
# the REQUIRES/XFAIL clause for individual tests. That's why we use the approach
# of external overrides.
#
# Prefer `xfail` for tests that fail reliably. Use `skip` for flaky tests.
# Test summaries call them "excluded".
